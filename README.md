# wave-file-specifications

http://www-mmsp.ece.mcgill.ca/Documents/AudioFormats/WAVE/WAVE.html

```bash

wget --mirror http://www-mmsp.ece.mcgill.ca/Documents/AudioFormats/WAVE/Samples.html
```
